# Secure Online Banking System

A full-stack banking web application built using Spring Boot and React.
The system supports user authentication, secure transactions, and account management.

## Tech Stack
- Backend: Spring Boot, Spring Security, JWT
- Frontend: React, Axios
- Database: MySQL
- Version Control: Git & GitHub

## Features
- User Registration & Login
- JWT-based Authentication
- Balance Management
- Fund Transfer
- Transaction History
- Admin Panel (future scope)
